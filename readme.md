# 应用开发指南

## 准备工作

+ 开通阿里云函数计算，
+ 下载serverless devs 命令行或者 desktop工具（详情查看官网 https://serverless-devs.com/zh-cn）
+ [配置秘钥信息](https://serverless-devs.com/zh-cn/docs/provider-config/alibabacloud.html)

## 快速开始
### 应用初始化
```
s init blindbox-game
```
### 应用部署

```
cd blindbox-game && s deploy
```

### 应用访问操作
根据返回的url 进行抽奖操作
